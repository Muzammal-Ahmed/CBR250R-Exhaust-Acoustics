# CBR250R M4 Exhaust Acoustic Characterization

Acoustic measurement and modeling of an M4 slip-on exhaust system 
on a 2012 Honda CBR250R, performed summer 2026.

## Project overview
- Calibrated SPL measurements across the full RPM range
- FFT spectral analysis identifying firing-order harmonics
- Muffler insertion loss characterization (M4 vs. straight-pipe)
- Transfer matrix method acoustic model validated against measurements
- Proposed geometry optimization targeting dominant noise frequencies

## Tools & methods
Python (NumPy, SciPy, Matplotlib) · Audacity · Dayton Audio iMM-6 mic
